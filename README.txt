README.txt
==========

This module provide an ability to set view mode for each individual entity.

The module compartible with Paragraphs module and allow to set nice and flexible Landing pages with the Paragraphs approach.


CONFIGURATION
-------------
 * Enable the module.

 * Add new field with of "Entity Display Field Display" type to the Entity that you need to manage view mode individually.

 * Don't forget to add and to align several display types for that entity.

 * Create an entity and set it's display type in the entity edit form.

 * Enjoy the result: this entity will be shown in that individual dislpay mode on the site public pages.
